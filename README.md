# klaytn Seed

## How to Install 

```
yarn global add truffle
yarn
```

## Test with truffle

```
yarn test
```
or
```
truffle test
```


## Set Secret file to contain Private Key

```
touch .secret

```

## Set Env for azure storage

```yml
AZURE_STORAGE_ACCOUNT_NAME=
AZURE_STORAGE_ACCOUNT_ACCESS_KEY=
AZURE_STORAGE_CONTRACT_CONTAINER_NAME=
```
